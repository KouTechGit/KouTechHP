授業の概要

本時では、数と式の分野から「因数分解」について学習しました。因数分解は、展開の逆の操作であり、整式をいくつかの整式の積の形に変形することを指します。この操作は、2次方程式の解法や式の計算において非常に重要な基礎技術となります。特に、2次3項式の因数分解（たすき掛け）や、置き換えを利用した因数分解の工夫について重点的に扱いました。因数分解ができるようになることで、より複雑な式の計算や方程式の解法への理解が深まります。

本時のポイント

  • 因数分解の基本公式：$a^2 - b^2 = (a+b)(a-b)$（平方の差）、$a^2 \pm 2ab + b^2 = (a \pm b)^2$（完全平方）、$a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)$（立方の和・差）を確実に覚えること。
  • 共通因数のくくり出し：まず共通因数があるかどうかを確認し、$ax + ay = a(x+y)$ のようにくくり出す。これが因数分解の第一歩。
  • 2次3項式の因数分解（たすき掛け）：$ax^2 + bx + c$ の形の式を因数分解する際は、$a$ と $c$ の約数の組み合わせを考え、たすき掛けで $b$ を作る方法を習得する。例：$2x^2 + 7x + 3 = (2x+1)(x+3)$
  • 置き換えを利用した因数分解：複雑な式では、一部分を別の文字で置き換えることで見通しが良くなる。例：$(x+1)^2 - 4(x+1) + 3$ で $x+1 = t$ とおくと $t^2 - 4t + 3 = (t-1)(t-3)$
  • 複数の文字を含む因数分解：2文字以上の式では、1つの文字について整理してから因数分解する。例：$x^2 - y^2 + 2x - 2y = (x-y)(x+y) + 2(x-y) = (x-y)(x+y+2)$
  • 因数分解の順序：①共通因数のくくり出し → ②公式の適用 → ③たすき掛け → ④置き換えの順で考えると効率的。
  • 板書例：$6x^2 - 7x - 3$ の因数分解：$6x^2 - 7x - 3 = (2x-3)(3x+1)$。たすき掛けで、$2 \times 1 + (-3) \times 3 = 2 - 9 = -7$ となり、$x$ の係数 $-7$ が作れることを確認。
  • プリント例題：$(x+2)^2 - 9$ の因数分解：$x+2 = A$ とおくと、$A^2 - 9 = (A+3)(A-3) = (x+2+3)(x+2-3) = (x+5)(x-1)$。平方の差の公式を利用。

復習問題

用語の確認

次の用語の意味を説明してください。

  1. 因数分解とは何か。
  2. 共通因数とは何か。
  3. たすき掛けとは何か。
解答を見る
  1. 整式をいくつかの整式の積の形に変形すること。
  2. 各項に共通して含まれる因数（共通因子）のこと。例：$ax + ay$ の $a$。
  3. 2次3項式 $ax^2 + bx + c$ を因数分解する際に、$a$ と $c$ の約数の組み合わせを考えて $b$ を作る方法。

公式の確認

次の公式を完成させてください。

  1. $a^2 - b^2 = $（　　）
  2. $a^2 + 2ab + b^2 = $（　　）
  3. $a^3 + b^3 = $（　　）
解答を見る
  1. $a^2 - b^2 = (a+b)(a-b)$
  2. $a^2 + 2ab + b^2 = (a+b)^2$
  3. $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$

基本計算問題

次の式を因数分解してください。

  1. $x^2 - 9$
  2. $x^2 + 6x + 9$
  3. $2x^2 + 5x + 2$
  4. $x^2 - 4xy + 4y^2$
解答を見る
  1. $x^2 - 9 = (x+3)(x-3)$
  2. $x^2 + 6x + 9 = (x+3)^2$
  3. $2x^2 + 5x + 2 = (2x+1)(x+2)$
  4. $x^2 - 4xy + 4y^2 = (x-2y)^2$

共通テスト形式

次の問題に答えなさい。

問題：$x^2 - 6x + 9$ を因数分解すると（　　）となる。

選択肢：① $(x-3)^2$　② $(x+3)^2$　③ $(x-3)(x+3)$　④ $(x-9)^2$

解答を見る

答え：① $(x-3)^2$

解説：$x^2 - 6x + 9 = x^2 - 2 \cdot 3x + 3^2 = (x-3)^2$ となり、完全平方の公式が適用できます。
